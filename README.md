# Security-seystem-using-a-Arduino-uno-R4

In this project I making security system for my room using a Arduino Peizo buzzer, a Ultrasonic Sensor and a transmitter and reciver. So far it is able to ring a peizo buzzer by using the tone function, which rings the peizo buzzer at 3000 hertz through pin 5 of the Arduino. If the following factors are true, if the door is open which it tells using a contact plate on the door, if the transmitter is not transmitting a signal to the ardiono and if the ultrasonic sensor detects something within 180 Cm. 

I am currently working on making it lock my room using a motor, some 3d printed gears and a rack and pinion.
